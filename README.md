# 🗳️ Online Voting System

A simple and secure Online Voting System developed using HTML, CSS, JavaScript, PHP, and MySQL. This project allows users to register, login, vote for candidates, and view results.

---

## 🚀 Features

- 🔐 User Registration & Login System  
- 🧑‍💼 Candidate Registration  
- 🗳️ Voting Functionality (One user, one vote)  
- 📊 Live Result Display  
- 🔒 Session-based Authentication  
- ⚙️ Admin Panel for Management  

---

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Server: Apache (XAMPP)
